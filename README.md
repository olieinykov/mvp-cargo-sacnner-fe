# MVP Cargo Scanner FE

React + Vite + TypeScript dashboard with shadcn-style components, Tailwind CSS, Formik + Zod forms and TanStack Query for server communication.

## Available scripts

- `npm run dev` — start development server
- `npm run build` — build for production
- `npm run preview` — preview production build

## Structure

- `src/components` — all shared components
- `src/components/ui` — shadcn-inspired UI primitives
- `src/components/layout` — layout components (sidebar, header)
- `src/components/audits` — Audits page components
- `src/lib/api` — server API helpers and TanStack Query hooks
